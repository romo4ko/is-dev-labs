# Описание датасета: Прогноз сердечно-сосудистых заболеваний

## Общая информация

Датасет предназначен для решения задачи бинарной классификации — предсказания наличия или отсутствия сердечно-сосудистых заболеваний (ССЗ) у пациентов на основе медицинских и демографических показателей.

## Описание датасета

Существует 3 типа входных данных в датасете:

- Объективный признак: фактическая информация;
- Признак обследования: результаты медицинского обследования;
- Субъективный признак: информация, предоставленная пациентом.

Все значения набора данных были собраны в момент медицинского обследования.

| Признак                               | Тип переменной           | Variable    | Тип значения                             |
|:-------------|:-------------|:-------------|:-------------|
| Возраст                               | Объективный признак      | age         | int (дни)                                |
| Рост                                  | Объективный признак      | height      | int (см)                                 |
| Вес                                   | Объективный признак      | weight      | float (кг)                               |
| Пол                                   | Объективный признак      | gender      | категориальный код                       |
| Систолическое артериальное давление   | Признак обследования     | ap_hi       | int                                      |
| Диастолическое артериальное давление  | Признак обследования     | ap_lo       | int                                      |
| Холестерин                            | Признак обследования     | cholesterol | 1: Норма (< 4 ммоль/л), 2: Повышенный (5 - 6.5 ммоль/л),  3: Высокий (> 7 ммоль/л)|
| Глюкоза                               | Признак обследования     | gluc        | 1: Норма (4.1 - 5.9 ммоль/л),  2: Повышенный (6 - 10 ммоль/л), 3: Высокий (> 10 ммоль/л)|
| Курение                               | Субъективный признак     | smoke       | бинарный                                 |
| Употребление алкоголя                 | Субъективный признак     | alco        | бинарный                                 |
| Физическая активность                 | Субъективный признак     | active      | бинарный                                 |
| Наличие или отсутствие сердечно-сосудистых заболеваний | Целевая переменная       | cardio      | бинарный                                 |
